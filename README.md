# View-Serializability Playground Backend
 

## Prerequisites

Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/)
- [NPM (Node Package Manager)](https://www.npmjs.com/)
 
## Development 
  To start the application, make sure to clone this repository and to install all the dependency needed. 
  
  Then to test it, you have to edit the .env file by adding your mysql string connection to the Database.
  
  Finally by running the command

     npm run start:dev

  The application will be accessible at http://localhost:3000 by default.
  
  Finally by going at http://localhost:3000/api# we have access to the Swagger and we can see all the features!
